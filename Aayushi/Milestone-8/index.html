Question-1

No one likes maths, and so do you. But your teacher doesn’t understand this and has given you the assignment to calculate the factorial of 50 numbers. Let us use the knowledge of programming we have gained so far to our benefit, write a function to calculate factorial of a number.


        
          <script>
              var num;
              num=prompt("Enter a number : ",num);
              function factorial(num){
                var fact=1;
                for(var i=1; i<=num; i++){
                  fact=fact*i;
                }
                return fact;
              }
              var fact=factorial(num);
              alert("Factorial of "+ num +" : "+fact);
          </script>
